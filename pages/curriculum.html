Curriculum

This page presents an overview of what knowledge student clubs should target during the year. The curriculum is not a list of all required information but more a invitation to learn.

  • Neurosciences basics
    • The central nervous system in brief
    • Principal regions of the brain, structurally and functionally
    • Electricity in the brain
    • Functional processes
      • Touch
      • Vision
      • Audition
      • Motor control
      • High-level processes
  • Brain-computer interfaces in general
    • Invasive / non-invasive
    • EEG, MEG, PET, SPECT, fMRI, fNIRS
  • Biosignal acquisition (focused on electrical signal)
    • Electrodes and impedance
    • Amplification
    • Analog signal processing
    • Digitalization / Data transmission
    • Security aspects
  • Software-related stuff
    • Connection with Device - communication protocol
    • Analysis algorithm
      • Time domain
      • Frequency domain
      • Event detection
      • Feature extraction
      • (Un)supervised learning
      • ...
    • Online Vs offline
    • Online
      • Sliding window analysis
    • Offline
      • Event-related / averaging
    • Digital signal processing
      • Hardware Vs Software
  • Experiments
    • EMG
    • ECG
    • Concentration / relaxation
    • Event-Related Potential (ERP)
    • P300
    • Emotions
    • ...
  • Extra curriculum
    • MEG
    • fNIRS

If you feel that something is missing from this curricul, please contact us.
